准备上线一个栏目：RCMS

Read (Code Thinking) Change Myself Style

准备上线的三个系列为：

> Java Puzzlers

> Java Concurrency in Practice

> Effective Java

  系列的原作者皆是Joshua Bloch,三本书都和Android没有直接的关键联系，但确是进阶代码范式编程的核心思维,很多优秀的Android开源架构都基于这些思维模式，所以总结出来。

Java Puzzlers格式：

> 谜题编号

> 核心知识点

> wrong 写法

> correct写法

Example：

谜题 1：

奇数性

核心：

写程序时要注意，判断一个数字是否为奇数的标准是，其能否被2整除，即余数是否为0，而不是”余数是否为1“。

WRONG：

return i % 2 == 1;

CORRECT：

    return i % 2 != 0;
or：
    return (i & 1) != 0;
